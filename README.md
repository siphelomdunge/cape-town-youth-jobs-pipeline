Cape Town Youth Jobs Data Pipeline is a data engineering project that collects, cleans, stores, and analyses youth job opportunity data.

The project uses Python to build the data pipeline for extracting and cleaning job data, and Java to build a backend system that can read and serve the cleaned data. The goal is to help young people understand which jobs, skills, and locations are most in demand in Cape Town.

This project demonstrates data engineering concepts such as ETL, data cleaning, database storage, SQL, backend development, and dashboard reporting.
